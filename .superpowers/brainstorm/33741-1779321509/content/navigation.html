Navigation Behavior

Auth Gate

If no Clerk session → redirect to sign-in. If signed in but no profile → create profile via Edge Function. If role mismatch → show error (guards can't open driver app).

Bottom Tabs

4 tabs each app. Active tab uses filled icon variant + container highlight + glow shadow. Inactive tabs use outlined variant at 60% opacity.

Top App Bar

Shared component: avatar (left), centered "UniPark" brand mark, notification bell with indicator dot (right). Glassmorphism backdrop-blur on both apps.
